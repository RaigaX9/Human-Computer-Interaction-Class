Submission Page

Users will be able to submit their own ideas of the robots that they've already made or want to see. This will all be done in Javascript where it will tell if the user is typing their information correctly. After they've submitted the proper information, it will take them to a "Thank You" page. All fields are required for submission except for uploading a file which is optional.

What idea(s) do you have for making your own robot? Or what kind of robot do you want to see being developed?

(Optional) You can either upload a document, an image, a video, or an image that's within the word document to explain what you made or ideas are.
