Conceptual Design:
The conceptual design highlights the main components that are going to be on the website.

Plan:
The plan for the website shows the type of medias such as images, videos, and other components that are going to be in there.

Structure:
The structure of the website shows the where all the links of the sections are going to be in the website and how they all relate with each other.

Multimedia Types:
Multimedia types are needed to provide certain functionalities to the website. This means that you need to install certain plug-ins for the browsers in order to make them work such as videos, images, audio, etc. For this website, the following will include certain multimedia types that are going to be in there:

  • CSS Files: Use the CSS files for each of the pages to format the website properly. This can also be done within the HTML file as well.

  • Videos: Here are two options that is going to be done.

    1. Embed a video from Youtube or any other websites similiar to it.

    2. If the video that I'm looking for cannot be found, video file within the server can be used as an alternative(Ex. .avi, .mp4, etc.).

  • Images: Use certain functions to identify each type of image files such as .jpg, .gif, .png, etc. Also, the image tag will allow the users to display the image. The type of image files will be the following:

    1. .jpg

    2. .jpeg

    3. .gif

    4. .png

  • Text Files: The text files will be used in plain text or in other formats.

  • HTML5 and CSS3 (possibility): Both of them may be used to create the website overall.
